# 🚀 HackFusion

HackFusion is an advanced coding practice platform designed to help developers improve their problem-solving skills through interactive challenges, contests, and detailed progress tracking.

## 📌 Features

- 🧠 Practice coding challenges across multiple difficulty levels.
- 🏆 Compete in real-time contests and leaderboards.
- 📊 Track your performance and growth over time.
- 🔎 Search problems by tags, difficulty, and categories.
- 💬 Discuss solutions and approaches with the community.
- 💻 Built with a modern tech stack for performance and scalability.
- 😎 AI Assistance

---

## 🛠️ Tech Stack

- **Frontend:** React.js / Vite / TailwindCSS
- **Backend:** Node.js
- **Database:** MongoDB,Redis
- **Authentication:** JWT-based auth
- **Other Tools:** Axios,Oauth,GitHub Auth

---

## ⚙️ Getting Started (Run Locally)

Follow the steps below to set up HackFusion on your local machine.

### ✅ Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (>= 16.x recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [MongoDB](https://www.mongodb.com/try/download/community) running locally or a cloud MongoDB URI (e.g., MongoDB Atlas)
git clone https://github.com/your-username/hackfusion.git
cd hackfusion
